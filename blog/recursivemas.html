RecursiveMAS: Latent-Space Multi-Agent Systems

What if agents didn't have to write essays to each other? RecursiveMAS replaces text handoffs with hidden-state telepathy, turning a team of LLMs into one unified computation graph. Here's how a 13M-parameter bridge module unlocks +8.3% accuracy, 2.4× speed, and −75.6% tokens.

01 The Problem

Today's multi-agent systems are like a team of brilliant researchers forced to communicate by writing letters, sealing them in envelopes, and waiting for the postal service. Every time Agent A wants to hand work to Agent B, it must decode its thoughts into human-readable text. Agent B reads that text and re-encodes it back into its own latent space. Two expensive trips through the vocabulary bottleneck — every single handoff.

This is framework collapse: the coordination overhead grows linearly with the number of agents and recursion depth. By the time you've chained three agents with two rounds of reflection, most of your compute budget has been spent on serialization rather than reasoning.

02 The Insight

What if agents could just think at each other? Instead of forcing latent reasoning through the narrow straw of vocabulary projection, RecursiveMAS passes hidden states directly from one agent to another. It treats the entire multi-agent team as a single recursive computation graph — one "mega-model" where each agent is a layer.

This is analogous to how ResNets skip connections work inside a single model, but scaled across model boundaries. The agents don't need to agree on a language because they speak in vectors.

03 The RecursiveLink Bridge

The magic is a tiny module: RecursiveLink (~13M parameters, just 0.31% of the total system). It's a 2-layer residual network with two modes:

  • Inner Link — folds an agent's own last-layer hidden state back into its input embedding space, creating an in-distribution "latent thought" stream inside the same agent.
  • Outer Link — projects latent thoughts from one agent to another, even across different hidden dimensions, so heterogeneous models can condition on each other directly.

The residual branch preserves original latent semantics; the module only learns the distributional shift. Base LLM parameters stay completely frozen. You're not retraining Qwen or Llama — you're learning a translation layer between their internal dialects.

04 The System Loop

Agents are arranged like layers in a recursive language model. The inner link feeds generated hidden states back as the next input. The outer link forwards latent thoughts to the next agent. The last agent feeds back to the first, closing the loop.

Only the final recursion round decodes to text. Everything intermediate stays in latent space. That means 10 rounds of deliberation between three agents costs roughly the same decode budget as a single forward pass through one agent — minus the tiny overhead of the RecursiveLink transforms.

05 How It Learns

Because the system is a closed loop, you can't train agents in isolation. RecursiveMAS uses a two-phase regime:

  • Inner loop: warm-start each agent's inner link via cosine regression, aligning latent thoughts toward the ground-truth answer embedding.
  • Outer loop: unroll the full system for n recursion rounds and minimize cross-entropy on the final textual output. Gradients backpropagate through the entire trace across agents and rounds.

The theoretical result is elegant: when token predictions are confident (low entropy), text-based recursive SFT suffers vanishing gradients because discrete token distributions saturate. RecursiveLink maintains near-constant gradient norms because it backpropagates through continuous hidden states rather than through argmax sampling.

06 Results

On standard multi-agent reasoning benchmarks, RecursiveMAS shows consistent gains across all four collaboration patterns:

+8.3% accuracy gain
2.4× inference speedup
−75.6% token reduction
$4.27 training cost

The speedup comes from dropping per-step vocabulary projection. Token reduction comes from eliminating intermediate decoding. The accuracy gain comes from preserving latent nuance that text serialization destroys. The low training cost comes from only updating the 13M-parameter RecursiveLink while keeping all base LLMs frozen.

07 Why It Matters for Agentra

If you're building agentic systems and wrestling with framework collapse — where multi-agent setups devolve into one long prompt — RecursiveMAS is a signal worth tracking. It treats coordination as an infrastructure layer rather than a text-formatting layer.

For Agentra's initiator-continuator architecture, the concepts transfer directly: lightweight micro-models (8–30M parameters) could serve as on-device initiators that pass latent context to cloud continuators via compact projection modules. The latent-space passing and whole-system backpropagation are techniques we can adapt when training coordination modules between micro initiators and cloud continuators — rather than relying purely on prompt engineering.

The paper's code is already public. The models are on HuggingFace. The whole system trains for less than a coffee. That's the kind of research-to-implementation gap we like to close.
