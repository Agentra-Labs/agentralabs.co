## Day Of — June 28, Saturday

### 2:00 PM — Setup Begins
**Team:** Ashwin + 1 helper
**Location:** Venue

- [ ] Arrive at venue
- [ ] Set up welcome table: sign, name badges, welcome packets
- [ ] Test projector/screen with laptop
- [ ] Set up camera on tripod (wide shot of presenter area)
- [ ] Test lapel mics with recording device
- [ ] Test ring light (position behind camera, not in presenter eyes)
- [ ] Set up backup phone on secondary tripod (safety angle)
- [ ] Arrange chairs: semi-circle or classroom style, 30 seats
- [ ] Place water bottles at every seat
- [ ] Set up "No photography past 8:30 PM" sign (subtle, by the door)
- [ ] Post WiFi password on whiteboard or sign

### 4:00 PM — Final Checks
- [ ] Sound check with each mic
- [ ] Camera framing check (presenter + screen visible)
- [ ] Lighting check (no harsh shadows on presenter face)
- [ ] Caterer arrival confirmed (food for 7:00 PM break + 8:30 PM dinner)
- [ ] Backup plan: identify quiet corner for phone backup recording

### 4:30 PM — Staff Briefing
**Team:** All staff (Ashwin + 2 helpers)

- [ ] Assign roles:
  - **Door / Check-in:** 1 person (names off list, hand badge + packet)
  - **AV / Recording:** 1 person (start/stop recording, mic battery check)
  - **Host / Timekeeper:** Ashwin (intro, transitions, keep time)
- [ ] Review emergency contacts
- [ ] Review "no recording after 8:30" policy enforcement
- [ ] Confirm dinner seating plan (intentionally mixed: researcher + builder at each table)

### 5:00 PM — Doors Open

**Music:** Low-volume ambient (lo-fi, jazz, or silence — test beforehand)
**Lighting:** Warm, not fluorescent

**Check-in process:**
1. Greet at door
2. Check name against confirmed list
3. Hand name badge + welcome packet
4. "WiFi password is on the card. Talks start at 5:30. Help yourself to chai."
5. If unexpected guest: "Are you [Name]?" If not on list: "We're at capacity, but let me check the waitlist."

### 5:00–5:30 PM — Pre-Event Mingling

- [ ] Chai and light snacks available
- [ ] Ashwin circulates, makes intros
- [ ] Intentionally introduce: "[Researcher], this is [Builder], they're working on [X]."
- [ ] 5:20 PM: Ask everyone to take seats
- [ ] 5:25 PM: AV person starts recording

### 5:30 PM — Welcome + Paper 1–3

**5:30–5:35** — Ashwin welcome
- "Welcome to Paper Club. 5 papers, 10 minutes each, then we argue."
- "Talks are recorded. Dinner is off the record. No phones after 8:30."
- "Introduce yourself to the person next to you if you haven't already."

**5:35–5:55** — Paper 1
- Presenter: [Name]
- Topic: [Paper Title]
- Timekeeper signals 2 min warning (gesture or card)
- Timekeeper signals hard stop at 10 min
- 5 min Q&A (moderated by Ashwin if needed)

**5:55–5:58** — Transition
- "2-minute break. Grab chai, stretch."

**5:58–6:18** — Paper 2
- Same format

**6:18–6:21** — Transition

**6:21–6:41** — Paper 3
- Same format

**6:41–7:00** — Extended Q&A / Discussion
- Open floor for 15 min
- Ashwin moderates: "Who disagrees with [point]? Who has a counter-example?"

### 7:00–7:15 PM — Break

- [ ] Caterer brings out dinner setup (or appetizers if dinner is at 8:30)
- [ ] People stretch, refill chai, use restroom
- [ ] AV person swaps SD cards / checks battery

### 7:15–8:30 PM — Paper 4–5 + Final Discussion

**7:15–7:35** — Paper 4
- Same format

**7:35–7:38** — Transition

**7:38–7:58** — Paper 5
- Same format

**7:58–8:30** — Open Discussion
- No structured Q&A
- Ashwin moderates lightly: "What surprised you? What was overhyped? What would you build from this?"
- This is where the real value happens

### 8:30 PM — Recording Stops / Dinner Begins

- [ ] AV person stops all recording
- [ ] Announce: "Recording stops now. Everything from here is off the record. Phones away."
- [ ] Caterer serves dinner
- [ ] Ashwin makes 2–3 intentional table intros
- [ ] Circulate, listen, don't dominate

### 8:30–10:00 PM — Dinner (Off the Record)

- [ ] No photos, no tweets, no recordings
- [ ] If someone pulls out phone: gentle reminder
- [ ] Staff stays available but doesn't hover
- [ ] Around 9:30: start subtle wind-down (lower music, dim lights slightly)

### 10:00 PM — Event Ends

- [ ] Thank everyone at door
- [ ] "Talks will be posted on YouTube this week. Session 2 applications open July 1."
- [ ] Collect any feedback verbally (no forms — too corporate)
- [ ] Staff cleanup: pack gear, check for lost items

## Emergency Protocols

| Scenario | Response |
|----------|----------|
| Presenter no-show | Ashwin presents their paper (15 min prep), or skip to 4 papers |
| AV failure | Use phone as primary, continue without recording if necessary |
| Caterer late | Send someone to nearest restaurant for emergency biryani |
| Over-capacity (35+ show up) | Add chairs, squeeze in. Don't turn away at door if physically possible. |
| Under-capacity (<20) | Combine tables, encourage mingling. Don't mention it. |
| Disruptive attendee | Quiet sidebar: "This is a curated room. Let's keep it constructive." |
| Power outage | Phone flashlights + keep talking. It's Kolkata — people roll with it. |
