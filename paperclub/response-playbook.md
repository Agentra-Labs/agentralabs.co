# Response Handling Playbook
## Agentra Paper Club — Outreach Operations

---

## Response Categories

When a reply comes in, tag it immediately in outreach-crm.csv under `response_type`.

| Code | Meaning | Action |
|------|---------|--------|
| **INTERESTED** | Wants to present or attend | Send detailed info + calendar hold within 2h |
| **MAYBE** | Interested but needs details / timing | Send detailed info + offer call. Follow up in 3 days. |
| **BUSY** | Can't make June 28 but wants future | Add to Session 2 pre-list. No follow-up needed. |
| **NO** | Not interested | Tag `rejected`. No follow-up. |
| **NO_REPLY** | No response after 7 days | Send follow-up 1. If still no reply after 14 days, send follow-up 2. Then archive. |
| **REFERRAL** | Can't come but suggests someone | Thank + ask for intro. Add referred person to CRM with `source: referral_from_[id]` |
| **SPONSOR** | Wants to sponsor instead of attend | Forward to sponsor-deck.html. Schedule 15-min call. |

## Operational Rules

1. **Response time:** Reply to all responses within 2 hours during waking hours (India time: 9 AM — 11 PM IST). If US-based contact, reply within 12 hours.
2. **Tone:** Personal, never corporate. Use first name. Reference their specific work if possible.
3. **No automation on replies:** Every response gets a human-written reply. Templates are starting points — customize 30%+ each time.
4. **Calendar holds:** For INTERESTED presenters, send calendar hold immediately. Use Calendly or Google Calendar invite.
5. **Cohort updates:** Every Thursday, send a "This week in Paper Club" email to all INTERESTED + MAYBE contacts with: confirmed presenters, paper shortlist teaser, new sponsor if any.
6. **Waitlist dignity:** If someone is waitlisted, tell them why and when they might get in. Don't ghost.
7. **Rejection grace:** If someone is rejected, send a kind note. Offer Session 2 early access.
